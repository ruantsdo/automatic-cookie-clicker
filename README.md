# 🍪 Cookie Clicker Bot

Este projeto foi desenvolvido em **Python** utilizando **Selenium** e o **Google Chrome Driver**.  
O objetivo é estudar e praticar automação com Selenium, criando um script que joga **Cookie Clicker**, comprando produtos e upgrades automaticamente.

📚 Este projeto foi feito apenas para fins de estudo e prática de Python e Selenium.

---

## 🚀 Funcionalidades
- Clica automaticamente no cookie principal.  
- Compra upgrades assim que disponíveis.  
- Compra produtos para aumentar a produção.  

---

## 🛠 Tecnologias utilizadas
- [Python](https://www.python.org/)  
- [Selenium](https://www.selenium.dev/)  
- [ChromeDriver](https://googlechromelabs.github.io/chrome-for-testing/#stable)  

---

## 📦 Pré-requisitos

Antes de rodar o projeto, você precisa ter instalado:

- Python 3.x  
- Pip (gerenciador de pacotes do Python)  
- Google Chrome  
- ChromeDriver compatível com a versão do seu navegador  

---

## ⚙️ Instalação e uso

1. Clone este repositório:
   ```bash
   git clone https://github.com/seu-usuario/cookie-clicker-bot.git
   cd cookie-clicker-bot
   
2. Instale as dependências:
   ```bash
   pip install -r requirements.txt

3. Baixe o ChromeDriver correspondente à versão do seu sistema em: https://googlechromelabs.github.io/chrome-for-testing/#stable

4. Extraia o arquivo e coloque o executável na raiz do projeto com o nome: chromedriver.exe

5.Execute o script:
  ```bash
  python cookieCliker.py
  ```
